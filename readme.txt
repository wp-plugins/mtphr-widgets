== Description ==

Custom pack of widgets including shortcode generators.

* Twitter
* Social
* Recent Comments
* Recent Posts
* Tabbed Posts
* Contact
* Post Navigation
* Collapse

### Advanced Info
Each widget includes a **shortcode generator**. Copy and paste the shortcodes into your posts to display the widget contents anywhere on your site.

Each widget also displays the widget ID of the widget. This can be helpful if you wish to use any of the filters built into the widgets.

== Installation ==

1. Upload `mtphr-widgets` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to your widgets page to use your new widgets!
